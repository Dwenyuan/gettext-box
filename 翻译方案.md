

# 新建项目流程
```mermaid
sequenceDiagram
    autonumber
    participant sys as 磁盘
    participant box as electron壳
    participant web as web端
    alt 新建项目流程
        alt 扫描项目
            sys->>+box: 扫描文件夹，获取信息
        else
            sys->>+box: 读取PO文件
        end
        box->>box: 生成translation对象
        box->>+web: 发送转换后的对象
        web->>web: 修改文案/合并翻译
        web->>-box: 保存
        box->>-sys: 保存成PO文件
    end
```
# 更新翻译流程（修改文案/合并翻译）
```mermaid
sequenceDiagram
    autonumber
    participant sys as 磁盘
   
    alt 读取已有翻译文件
        sys->>view: 读取excel文件
        activate view
    else
        sys->>view: 读取JSON文件
    else
        sys->>view: 读取message.po文件
    else
    end
    alt web端操作
        alt 合并已有文件操作
            activate view
            view->>view: 转换为translation对象
            deactivate view
            view->>action: 发送对象
            deactivate view

            action->>epic: 发送对象
            epic->>store: 发送对象
            activate store
            note over store: 合并大对象速度慢，需要优化
            store->>store: 合并对象
            store->>view: 订阅变化
            deactivate store
        end
        alt 手动修改文案
            view->>action: 手动修改
            action->>epic: 发送对象
            epic->>store: 发送对象
            activate store
            store->>store: 合并对象
            store->>view: 订阅变化
            deactivate store
        end
    end
    view->>sys: 保存文件
```
